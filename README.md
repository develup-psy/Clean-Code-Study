## 📅 스터디 진행 방식

### 🔄 진행 일정
- **스터디 정기 발표 및 회의** : 매주 **목요일 방과 후** 진행
- **공부 자료 업로드 마감** : 매주 **수요일 24시까지**

### 📌 진행 방식
1. 매주 정해진 분량을 각자 공부한 후 **GitHub에 업로드** (마크다운 형식)
2. 챕터별 **담당자가 정리 내용을 발표**하고, 팀원 전체와 함께 리뷰 진행
3. **스터디원 발표 순서**:
   - 🦌 이주미 → 🦌 박양하 → 🦅 김운경 → 🐈 김여진 → 🦁 박성용

<br>
